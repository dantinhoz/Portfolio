# 💼 Portfólio Pessoal
Bem-vindo ao meu portfólio profissional! Este é um site estático que apresenta meus projetos, habilidades e informações de contato.

## 📋 Descrição
Um portfólio moderno e interativo desenvolvido com **HTML5** e **CSS3**. O site apresenta uma interface limpa, com futuro suporte a modo escuro/claro, navegação intuitiva e showcase de projetos de desenvolvimento.

## 🎯 Funcionalidades
- **Seção de apresentação** - Introdução profissional com foto e redes sociais
- **Links para redes sociais** - GitHub, Instagram, LinkedIn e contato

## 🎯 Funcionalidades futuras
- **Navegação responsiva** - Menu de navegação suave e funcional
- **Modo noturno/claro** - Alternância entre temas com um clique
- **Galeria de tecnologias** - Exibição das tecnologias dominadas
- **Portfólio de projetos** - Cards interativos com descrição, tecnologias e links
- **Totalmente responsivo** - Adaptado para desktop, tablet e mobile

## 💻 Tecnologias Utilizadas
- **HTML5** - Estrutura semântica
- **CSS3** - Estilização e design
- **Font Awesome** - Ícones profissionais
- **Git** - Controle de versão

## 🚀 Como Usar
1. Clone o repositório:
```bash
git clone https://github.com/seu-usuario/portfolio.git
cd portfolio
```

2. Abra o arquivo `index.html` diretamente no navegador

3. Ou utilize um servidor local (recomendado):
```bash
# Com Python 3
python -m http.server 8000

# Com Node.js (live-server)
npx live-server
```

Então acesse `http://localhost:8000` no navegador.
